# 音頻檔案說明

將以下音頻檔案放置在此資料夾中：

## 背景音樂
- `background_music.mp3` - 主要背景音樂

## 音效檔案
- `line_clear.mp3` - 消除行時的音效
- `piece_drop.mp3` - 方塊落地音效
- `piece_rotate.mp3` - 方塊旋轉音效
- `game_over.mp3` - 遊戲結束音效

## 音頻格式建議
- 格式：MP3 或 WAV
- 品質：中等品質即可 (128-192 kbps)
- 長度：背景音樂建議 30-60 秒循環播放
- 音效建議控制在 1-3 秒內

## 免費音頻資源
可以從以下網站取得免費的遊戲音效：
- freesound.org
- zapsplat.com
- mixkit.co
- opengameart.org

注意：使用音頻時請確認授權條款。
