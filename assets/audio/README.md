# 音頻檔案說明

將以下音頻檔案放置在此資料夾中：

## 背景音樂
- `background_music.mp3` - 主要背景音樂

## 音效檔案
- `line_clear.mp3` - 消除行時的音效
- `piece_drop.mp3` - 方塊落地音效
- `piece_rotate.mp3` - 方塊旋轉音效
- `game_over.mp3` - 遊戲結束音效

## 符文音效檔案
- `rune_angels_grace.mp3` - 天使恩典符文音效
- `rune_blessed_combo.mp3` - 祝福連擊符文音效
- `rune_dragon_roar.mp3` - 龍吼符文音效
- `rune_element_morph.mp3` - 元素變形符文音效
- `rune_flame_burst.mp3` - 火焰爆裂符文音效
- `rune_gravity_reset.mp3` - 重力重置符文音效
- `rune_thunder_strike_left.mp3` - 雷霆一擊（左）符文音效
- `rune_thunder_strike_right.mp3` - 雷霆一擊（右）符文音效
- `rune_time_change.mp3` - 時間扭曲符文音效
- `rune_titan_gravity.mp3` - 泰坦重力符文音效

## 音頻格式建議
- 格式：MP3 或 WAV
- 品質：中等品質即可 (128-192 kbps)
- 長度：背景音樂建議 30-60 秒循環播放
- 音效建議控制在 1-3 秒內
