# 如何識別惡魔方塊 vs H型稀有方塊

## 🎯 快速識別指南

你遊戲中有 **兩種特殊方塊**，它們看起來可能很相似，但實際上完全不同：

---

## 🌟 H型稀有方塊（經常出現）

### 視覺特徵
- **顏色**: 霓虹青檸色（黃綠色）`#CCFF00`
- **格數**: 7 格
- **形狀**: 固定的 H 型
- **大小**: 3×3

### 形狀圖示
```
█ █
███
█ █
```

### 出現規則
- **觸發條件**: 每 30 個方塊自動出現 1 次
- **分數要求**: 無（自動觸發）
- **出現時機**: 遊戲開始後很快就會出現（大約放置 30 個方塊後）
- **頻率**: 很頻繁（約每分鐘 1-2 次）

### 特殊能力
- ✅ **可以旋轉**（4 個方向）
- ❌ 鎖定後**沒有**分數加成
- ❌ 不計入惡魔方塊統計

### 控制台日誌
```
[RareBlockInterceptor] 🌟 Generating rare block: H (7 cells, YELLOW-GREEN color)
```

---

## 🔥 惡魔方塊（罕見 Boss 方塊）

### 視覺特徵
- **顏色**: 金色 `#FFD700`（更偏橙黃色）
- **格數**: 10 格
- **形狀**: 隨機不規則形狀（每次不同）
- **大小**: 最多 5×5

### 形狀範例
```
  ███
█████
  ███
  █
```
（每次生成都不同！）

### 出現規則
- **觸發條件**: 達到特定分數門檻時自動出現
- **分數要求**:
  - 第 1 次: 10,000 分
  - 第 2 次: 22,974 分
  - 第 3 次: 37,372 分
  - ...（最多 15 次）
- **出現時機**: 通常在遊戲中後期才會出現
- **頻率**: 罕見（取決於你的遊玩技術）

### 特殊能力
- ❌ **不能旋轉**（任何旋轉操作都會失敗）
- ✅ 鎖定後啟動 **3× 分數加成**（持續 10 秒）
- ✅ 計入惡魔方塊統計

### 控制台日誌
```
═══════════════════════════════════════════
🔥🔥🔥 DEMON BLOCK TRIGGERED! 🔥🔥🔥
Current Score: 10000
Next piece will be: DEMON (10 cells, GOLD color)
═══════════════════════════════════════════

[DemonSpawnManager] Spawn #1 triggered at score 10000 (threshold: 10000)
[DemonSpawnManager] Next threshold: 22974
```

## 🔍 如何確認你看到的是哪一種？

### 方法 1：查看控制台日誌

**如果看到這個 = H型稀有方塊：**
```
[RareBlockInterceptor] 🌟 Generating rare block: H (7 cells, YELLOW-GREEN color)
```

**如果看到這個 = 惡魔方塊：**
```
🔥🔥🔥 DEMON BLOCK TRIGGERED! 🔥🔥🔥
```

### 方法 2：數格子數量

在 Next Piece 預覽中數一下方塊有幾格：
- **7 格** = H型
- **10 格** = 惡魔

### 方法 3：嘗試旋轉

當方塊下落時，按旋轉按鈕：
- **能旋轉** = H型
- **不能旋轉** = 惡魔

### 方法 4：查看當前分數

- **分數 < 10,000** → 不可能是惡魔方塊，一定是 H型
- **分數 ≥ 10,000** → 可能是惡魔方塊（檢查日誌確認）

### 方法 5：觀察鎖定後的效果

鎖定方塊後，看控制台：
- **沒有特別訊息** = H型
- **顯示分數加成啟動** = 惡魔
  ```
  [GameLogic] Demon block locked! Score multiplier activated (×3 for 10s)
  [GameState] Score multiplier activated! Duration: 10s
  ```

## 🎮 實際遊玩範例

### 場景 1：遊戲剛開始
```
分數: 500
放置方塊數: 25
出現: 黃綠色 7格 H型方塊
→ 這是 H型稀有方塊 ✓
```

### 場景 2：遊戲中期
```
分數: 9,800
放置方塊數: 120
出現: 黃綠色 7格 H型方塊
→ 這是 H型稀有方塊 ✓（還沒到 10,000 分）
```

### 場景 3：首次達標
```
分數: 10,050
放置方塊數: 150
控制台: 🔥🔥🔥 DEMON BLOCK TRIGGERED! 🔥🔥🔥
出現: 金色 10格 不規則方塊
→ 這是惡魔方塊！✓
```

### 場景 4：可能的混淆
```
分數: 10,200
放置方塊數: 180
出現: 黃綠色 7格 H型方塊
控制台: [RareBlockInterceptor] 🌟 Generating rare block: H
→ 這還是 H型！（惡魔方塊已經觸發過了，下一次要等 22,974 分）
```

## ⚠️ 常見誤解

### 誤解 1：「黃色方塊 = 惡魔方塊」
❌ **錯誤**：H型也是黃色系（黃綠色）
✅ **正確**：惡魔方塊是純金色 + 10格 + 隨機形狀

### 誤解 2：「大方塊 = 惡魔方塊」
❌ **錯誤**：H型也比標準方塊大（7格 vs 4格）
✅ **正確**：惡魔方塊是 10 格，H型是 7 格

### 誤解 3：「出現特殊方塊 = 觸發惡魔」
❌ **錯誤**：H型每 30 個方塊就出現
✅ **正確**：惡魔方塊只在分數門檻時出現，且控制台會有明確提示

---

## 🧪 測試建議

### 如果你想確認惡魔方塊系統是否正常：

1. **開始新遊戲**
2. **觀察控制台**（保持控制台可見）
3. **快速刷分到 10,000**（使用 4 行消除）
4. **等待控制台出現：**
   ```
   🔥🔥🔥 DEMON BLOCK TRIGGERED! 🔥🔥🔥
   ```
5. **確認 Next Piece 預覽顯示 10 格方塊**
6. **嘗試旋轉 → 應該不能旋轉**
7. **鎖定後確認分數加成啟動**

如果以上都正確，惡魔方塊系統就是正常的！

---

## 📞 仍然不確定？

請查看控制台日誌，並回答以下問題：

1. 當前分數是多少？（< 10,000 = 不可能是惡魔）
2. 控制台是否顯示 `🔥🔥🔥 DEMON BLOCK TRIGGERED!`？
3. 方塊有幾格？（數一下）
4. 方塊能旋轉嗎？

**如果 1=<10000 或 2=否 或 3=7格 或 4=能，那就是 H型稀有方塊，不是惡魔方塊。**
